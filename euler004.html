Project Euler - Problem #004

Aufgabe

Lösung der vierten Problemstellung des Projekts Euler.

Ein Palindrom ist eine Zahl wie bspw. 101, die von vorn als auch hinten gelesen den gleichen Wert hat. Das größte Palindrom des Produktes von zwei zwei-stelligen Zahlen ist 9009 = 91 * 99.

Gesucht ist das größte Palindrom von zwei drei-stelligen Zahlen.
